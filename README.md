## Авторизация и права доступа

- JWT-авторизация: библиотеки `djangorestframework-simplejwt`.
- Эндпоинты:
  - POST /api/users/ — регистрация пользователя (доступен без авторизации).
  - POST /api/token/ — получение пары access/refresh токенов.
  - POST /api/token/refresh/ — обновление access-токена.

### Права

- Все эндпоинты, кроме регистрации и получения токенов, закрыты `IsAuthenticated`.
- Группа `moderators`:
  - может смотреть и редактировать любые курсы и уроки;
  - не может создавать и удалять курсы и уроки.
- Обычные пользователи (не модераторы):
  - видят/редактируют/удаляют только свои курсы и уроки (поле `owner`).

### Профиль

- Авторизованный пользователь может смотреть любой профиль.
- Редактировать и удалять можно только свой профиль.
- При просмотре чужого профиля не возвращаются фамилия и история платежей.
